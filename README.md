# Sakura Clang

Custom Clang 24.0.0 compiler for building Linux kernels and custom Sakura Kernel.

## Features
- Based on LLVM/Clang 24.0.0
- Optimized for Linux kernel builds
- Includes custom optimization flags
- Built using Neutron Clang build scripts
- Created for the personalization of the Sakura Kernel.
- Specifically designed for Sakura Kernel
