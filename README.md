# Sakura Clang

A custom Clang 24.0.0 compiler specifically designed for building Linux kernels and the custom Sakura Kernel.

## Features
- Based on LLVM/Clang 24.0.0
- Optimized for Linux kernel builds
- Includes custom optimization flags
- Built using **Neutron Clang** build scripts, adapted and modified for Sakura Clang
- Specifically designed for Sakura Kernel
- Includes LLD 24.0.0
